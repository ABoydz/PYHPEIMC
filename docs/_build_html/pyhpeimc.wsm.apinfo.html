pyhpeimc.wsm.apinfo.get_ap_info(ipaddress, auth, url)¶

function takes input of ipaddress to RESTFUL call to HP IMC :return:dictionatiry contraining the Access Point information for one AP as specified by the ipaddress as known by HPE IMC Wireless Services Manager module.

pyhpeimc.wsm.apinfo.get_ap_info_all(auth, url)¶

function takes no input to RESTFUL call to HP IMC :return:list of dictionatires contraining all Access Point information as known by HPE IMC Wireless Services Manager module.
